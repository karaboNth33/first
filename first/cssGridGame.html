Welcome to Spider Grid, where you write CSS code to catch your prey! Highlight only the areas that have insects by using the grid-column-start property.

For example, grid-column-start: 3; will highlight the area starting at the 3rd vertical grid line, which is another way of saying the 3rd vertical border from the left in the grid.
